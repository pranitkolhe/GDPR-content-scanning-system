# Project Setup
👉Project setup

1️⃣ Backend setup

cd backend
npm install
npm run dev

2️⃣ Python setup

cd ai_service
python -m venv .venv
source .venv/bin/activate   # macOS / Linux
.venv\Scripts\activate    # Windows

pip install -r requirements.txt
uvicorn app:app --port 8001 --reload

3️⃣ Frontend setup

cd frontend
npm install
npm run dev
